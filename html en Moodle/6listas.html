Listas Ordenadas

Son aquellas que nos muestran los elementos de la lista en orden

Sus elementos van precedidos de números o letras

Los elementos de las listas ordenadas van entre las etiquetas <ol> y </ol>

Cada uno de los elementos va entre las etiquetas <li> y </li>

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
  4. .....

Podemos cambiar el número de inicio de una lista con el atributo start

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
  4. .....

Podemos invertir el orden de la lista con el atributo reversed

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
  4. .....

Listas desordenadas

Nos sirven para mostrar elementos sin ningún orden, precedidos por una viñeta

Los elementos de las listas desordenadas van entre las etiquetas <ul> y </ul>

  • Primer elemento
  • Segundo elemento
  • Tercer elemento
  • .....

Listas de definiciones

Sirven para montar listas con la estructura valor y definición

Los elementos de las listas de definiciones van entre las etiquetas <dl> y </dl>

Los elementos van entre las etiquetas dt para el valor y dd para la definición

Término 1
Definición 1
Término 2
Definición 2
Término 3
Definición 3

Listas anidadas

Podemos hacer que el elemento de una lista sea otra lista cuyos elementos sean otras listas

Las anidaciones pueden ser de cualquier tipo de lista, sin límite de anidación

  • Carnívoro
    • Tigre
    • Pantera
    • Cocodrilo
  • Herbívoro
    • Oveja
    • Vaca
  • Omnívoro
    • Oso
    • Gallina
