Las imágenes en HTML

La etiqueta para poner una imagen en una página web es <img src = "ruta de la imagen" />

Esta etiqueta es un elemento vacío, no contiene texto ni etiqueta de cierre

Tiene un atributo obligatorio src

El atributo src contiene la ruta dónde se encuentra la imagen

Esta ruta puede ser relativa o absoluta

El nombre de la imagen tiene que ir con su tipo de archivo

Es muy importante de cara al SEO ponerle a las imágenes un nombre descriptivo

La mayoría de las imágenes tienen derechos de autor

Solo debes mostrar imágenes tuyas, con permiso del autor o libres de derechos

Cada imagen que se añade a la web incrementa el peso de la misma y por tanto su tiempo de descarga

El atributo alt

Su valor debe ser una descripción de la imagen

Se utiliza como alternativa a la imagen, para cuando esta no pueda ser mostrada

Una imagen puede que no se muestre por diversos motivos:

  • El usuario tiene discapacidad visual y utiliza un lector de pantalla
  • La ruta esté mal escrita
  • El navegador no muestra imágenes. Hay quien utiliza navegadores de solo texto (lynx)
  • Para los motores de búsqueda
  • Porque el usuario ha desactivado el uso de imágenes para ahorrar ancho de banda

Atributos width y height

Se utilizan para especificar la anchura y altura en que se muestra la imagen

Es conveniente utilizar solo uno de los dos para no deformar la imagen

Si utilizamos solo uno, el otro lo calcula el navegador guardando la proporción

Se suele utilizar la anchura: width

Atributo title

Este atributo proporciona información de ayuda al parar el ratón sobre la imagen

Los elementos figure y figcaption

El elemento figure representa un contenido independiente.

Generalemente es una imagen, un diagrama, un fragmento de código...

El elemento figcaption representa un subtítulo o leyenda asociado al elemento padre figure

El elemento figcaption puede ir como primer o último hijo de figure

Ejemplo de una imagen con figure y figcaption

El elemento figcaption vincula semánticamente el texto que contiene con el elemento contenido en figure

Favicon

El favicon es el icono que aparece en la pestaña del navegador o en la lista de marcadores.

El favicon es una imagen de tamaño reducido (16x16 px) y formato .ico

Algunos navegadores admiten formato PNG y Firefox admite SVG

La etiqueta para el favicon se pone dentro del head con una etiqueta link y los siguientes atributos:

  • rel = "icon"
  • type="image/tipo de imagen"
  • href="ruta a la imagen"

Si se incluye un archivo favicon.ico en la carpeta raiz del sitio web, los navegadores utilizan automáticamente esa imagen como favicon de todas las páginas del sitio sin que sea necesario incluir la etiqueta link

La etiqueta link es necesaria si se quieren utilizar otros formatos de imagen, guardarlo en otras carpetas o utilizar diferentes favicon para diferentes páginas

Con la utilización de navegadores móviles se ha complicado un poco el uso del favicon
